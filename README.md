# LyricsNotify

用于在 ClassIsland 平台上以丰富形式提醒显示歌词的插件。

> [!WARNING]
> 这篇 READEME 完全由 copilot 生成，可能有不准……

## 功能简介

`LyricsNotify` 使用户能够通过提醒方式同步播放本地音频并显示对应歌词（LRC 文件），适合需要在特定场景下自动化展示歌曲片段和歌词的需求。插件可自定义音频文件和歌词文件。

## 主要特性

- **歌词提醒**：解析并展示本地 LRC 歌词文件，配合本地音频同步滚动显示。
- **本地音频播放**：支持 MP3、WAV 等常见格式的本地音频作为提醒伴奏。
- **可自定义提醒内容**：支持设置提醒标题、遮罩文本、显示时长、动画效果等。
- **灵活的自动化调用**：可通过自动化动作（Automation）或指定事件调用歌词提醒。

## 使用说明

### 1. 提前准备

- 安装并启用`LyricsNotify`插件。
- 准备好本地音频文件（支持 mp3/wav）和对应的 LRC 歌词文件。

### 2. 配置提醒

1. 在 ClassIsland 平台【应用设置】->【插件】->`LyricsNotify`插件页面，找到设置面板。
2. 点击“选择音频文件”按钮，指定音频文件。
3. 点击“选择歌词文件”按钮，指定对应的 LRC 歌词文件。
4. 可调整**遮罩时长**（提醒标题显示时间）、**提醒内容**（自定义标题文字）、**播放音量**、**动画效果**等参数。
5. 可选择是否忽略歌词中的空行。
6. 支持自定义歌词滚动刷新频率，以适配不同性能和显示需求。

### 3. 触发自动化

- 可通过创建**自动化任务**，调用“显示歌词提醒”动作，实现定时或事件自动推送歌词展示。
